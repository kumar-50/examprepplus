**Design Philosophy:**
- **Simple** - No clutter, clear actions
- **Trustworthy** - Clear discount breakdown
- **Accessible** - Keyboard support, screen readers
- **Delightful** - Green success states, smooth interactions
- **Professional** - Matches existing brand/design system

This UI converts better than visible discount plans because:
1. Reduces decision paralysis (3 vs 5 options)
2. Makes users feel smart for having a code
3. Creates urgency (limited uses)
4. Clean, professional appearance
