# 06 Landing Page Content Expansion

## Objective

Add feature grid, problem/solution narrative, pricing teaser, responsive enhancements.

## Steps

1. Features grid (6 items) using shadcn `card`.
2. Problem vs Solution section (two-column at md breakpoint).
3. Pricing teaser: Free vs Premium summary with upgrade CTA.
4. Add FAQ accordion (reuse shadcn accordion).
5. Accessibility pass: headings hierarchy, alt text placeholders.

## Acceptance Criteria

- All sections scroll smoothly; semantic landmarks present.
- Mobile layout readable (no clipped text).

## Best Practices

- Use Tailwind utilities for spacing; avoid deep custom CSS.
