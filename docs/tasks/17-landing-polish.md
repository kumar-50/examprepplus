# 17 Landing Page Polish

## Objective

Finalize marketing page with SEO, performance, and accessibility improvements.

## Steps

1. Add meta tags (title, description, keywords) + Open Graph + Twitter card.
2. Generate OG image (static or dynamic).
3. Optimize images (next/image, proper sizes).
4. Add structured data (JSON-LD) basic organization/app.
5. Lighthouse pass (target >90 Performance/Accessibility).

## Acceptance Criteria

- Lighthouse scores meet targets.
- No missing alt attributes.

## Best Practices

- Avoid heavy JS in landing; minimal client components.
